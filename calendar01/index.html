원본 소스에서 바뀐 부분

  • 해당 월의 이전달, 다음달의 날짜를 미리 보여준다.
  • 이전달, 다음달의 날짜는 흐린 회색으로 표시한다.
  • ID로 작업된 CSS 를 Class 로 변경해서 여러개의 달력에 동일한 CSS 를 할당 할 수있고,
    개별적으로 CSS 를 할당할 수도 있다.

Calendar - Default

kCalendar('kCalendar',{
	callBack:show
});

function show(_date){
	console.log('show : ', _date)
}

Calendar - Set Date

kCalendar('calendar2',{
	initDate:'2016-05-16',
	callBack:show
});

function show(_date){
	console.log('show : ', _date)
}
